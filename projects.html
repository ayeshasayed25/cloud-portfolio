Project 2 – Terraform AWS Production Infrastructure

Project Overview

Designed and deployed a production-style AWS infrastructure environment using Terraform Infrastructure as Code (IaC). The project demonstrates enterprise cloud networking, security controls, compute deployment, and private workload management.

Infrastructure Components Completed

  • Custom AWS VPC (10.0.0.0/16)
  • Public Subnets across Availability Zones
  • Private Subnets across Availability Zones
  • Internet Gateway
  • Public Route Tables
  • Private Route Tables
  • Route Table Associations
  • Security Groups
  • Bastion Host EC2 Instance
  • Private Application EC2 Instance
  • Elastic IP
  • NAT Gateway
  • Private Subnet Internet Access
